## Project Overview
HandsMen Threads is a Salesforce CRM project developed for a retail clothing brand to manage customers, sales, and inventory efficiently using Salesforce automation.

## Key Features
- Customer and sales data management using custom objects
- Loyalty program implementation to improve repeat purchases
- Stock alert automation to notify managers when inventory is low
- Automated email notifications using email templates and scheduled flows
- Reports and dashboards for sales and customer tracking

## Salesforce Concepts Used
- Custom Objects and Fields
- Object Relationships
- Profiles, Roles, and Permission Sets
- Salesforce Flows and Validation Rules
- Reports and Dashboards
- Data Management (Import/Export)
